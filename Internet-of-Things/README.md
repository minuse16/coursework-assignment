# Internet of Things

Mini projects and assignments related to IoT systems, sensors, and smart automation.

## Featured Project
### Automatic Light Control System

Developed an automatic lighting control system using a light sensor to turn lights on/off based on ambient brightness.

## Features
- Automatic light detection using sensor
- Auto ON/OFF lighting system
- Data logging connected with Google Sheets
- Remote monitoring and control through Arduino Dashboard

## Technologies Used
- C++
- Arduino
- Sensors
- Google Sheets Integration
- Arduino Dashboard
